API Documentation

RESTful API for browser extension integration

Authentication

Include your API key in the Authorization header:

Authorization: Bearer YOUR_API_KEY
GET /api/check_domain_status/

Check domain security status

Parameters:

  • domain (string, required) - Domain to check (e.g., example.ng)

Response:

{
  "domain": "example.ng",
  "status": "secure",
  "threat_level": "low",
  "details": {
    "dns_changes": 0,
    "last_check": "2025-01-15T10:30:00Z"
  }
}
POST /api/domains/add/

Add domain to monitoring

Body:

{"domain": "example.ng"}

CORS Configuration

For browser extensions, ensure CORS headers allow extension origins. Contact support for whitelist requests.
